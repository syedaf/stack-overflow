# stack-overflow

A Next.js project with TypeScript, ESLint, and Prettier pre-configured.

## Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

## Scripts

- `npm run dev` - Development server
- `npm run build` - Build for production  
- `npm run lint` - Run ESLint
- `npm run format` - Format with Prettier
